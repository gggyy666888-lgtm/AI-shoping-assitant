# AI Shopping Assistant

智能购物决策助手移动端网页 Demo。项目使用 React、Vite 和本地 mock 数据，把“提出晚餐需求 → AI 生成方案 → 拆解原始食材 → 选择商品 → 优化购物车”做成可点击的交互流程。

## 当前进度

第一阶段已建立项目基础：五个页面路由、移动端页面外壳、底部导航、统一设计变量、mock 数据和素材入口。详细页面 UI 会在后续阶段逐页完成。

## 页面路由

- `/`：AI 助手首页
- `/understanding`：AI 需求理解页
- `/plan`：晚餐方案页
- `/product`：西红柿商品详情页
- `/cart`：智能购物车页

## 目录说明

- `reference/`：按页面流程排序的五张最终参考图。
- `assets/`：预留给机器人、蔬菜、鸡蛋、排骨等独立素材，暂时留空。
- `src/components/`：可以在多个页面复用的组件。
- `src/pages/`：五个路由对应的页面入口。
- `src/data/`：固定需求、菜品、食材和优惠 mock 数据。
- `src/config/assets.js`：后续替换本地图片时的统一入口。
- `src/styles/`：颜色、圆角、间距和全局样式。
- `docs/design-analysis.md`：参考图结构、组件与数据规划。

## 本地运行

1. 在本目录安装依赖：`npm install`
2. 启动开发预览：`npm run dev`
3. 打开终端显示的本地地址。

项目设计基准宽度为 390px，适配常见手机宽度；在桌面浏览器中会居中显示，最大宽度为 430px。
